// Function: An array-based queue.



Prompt:

Write a C program to implement a queue using an array. The queue supports the following operations:
a. size(queue): returns the number of items in the queue.
b. enqueue(queue, item): adds the item to the end of the queue.
c. dequeue(queue): removes and returns the item from the front of the queue.


Usage:

test.c includes a few pre-defined test cases. Run the source code to see the results.
